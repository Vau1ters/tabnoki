# インストール方法

## タブノキのビルド

```sh
$ cd tabnoki
$ npm i
$ npm run build
```

## chrome からの読み込み方

1. アドレス欄に `chrome://extensions`と入力する
1. 「パッケージ化されていない拡張機能を読み込む」ボタンを押す
1. `npm run build` で生成された `./build` ディレクトリを選択
1. 拡張を有効化する
